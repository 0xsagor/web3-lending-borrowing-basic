# Web3 Lending and Borrowing Basic

This project helps beginners learn how crypto lending platforms work.

You will learn:
- What lending and borrowing is
- How to connect a wallet
- How to deposit tokens
- How to borrow tokens
- How to check loan status

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-lending-borrowing-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage  
- app.js = Frontend logic  
- lending.js = Lending contract functions  
- web3.js = Blockchain connection  
- deploy.js = Contract deploy script  
- config.js = Network settings  
- server.js = Local server  
- package.json = Project info  
- utils.js = Helper functions  
- .env.example = Environment example  

## Features
- Connect wallet  
- Deposit tokens  
- Borrow tokens  
- Repay loan  
- Show loan balance  

## Learning Goal
This project explains how decentralized lending platforms work.
